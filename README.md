RFC3339
=======

Python module for that format dates according to the [RFC 3339](http://www.faqs.org/rfcs/rfc3339.html).

Import of the PyPI module published by Henry Precheur <henry@precheur.org>.

See https://pypi.python.org/pypi/rfc3339/

Installation from Github
========================

```pip install https://github.com/gnrfan/rfc3339/zipball/master```

Add this line to your ```requirements.txt``` file:

```-e git+https://github.com/gnrfan/rfc3339.git@HEAD#egg=rfc3339```

Then just run:

```pip install -r requirements.txt```.
